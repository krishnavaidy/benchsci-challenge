About

  • Xml files of all publications are stored on server.
  • Json files of processed publications with pmcid, figure uri, figure text, and co-occurrences are stored in MongoDB.
  • When pmcid is queried on page, MongoDB is queried to see if corresponding json file is stored, and if not, xml file of the publication is processed and json file is stored in MongoDB.
  • Results: figure uris and co-occurrences are showed on result page.
  • Json and csv files with all the data is currently not served, but code tp generate json file is on github.
